# 📚 Como acompanhar as aulas

Este repositório está organizado por **branches**, onde cada uma representa uma aula com seu respectivo código completo.

## 🔀 Estrutura das branches

- **main**  
  Contém a **estrutura inicial do projeto**.

- **demais branches**  
  Cada branch corresponde a uma aula específica, contendo o código exatamente como foi desenvolvido naquele ponto.

## 🚀 Passo a passo

Siga os passos abaixo para acompanhar corretamente:

1. Clone o repositório:
   ```bash
   git clone <url-do-repositorio>
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd <nome-do-projeto>
   ```

3. Liste todas as branches disponíveis:
   ```bash
   git branch -a
   ```

4. Acesse a branch da aula desejada:
   ```bash
   git checkout <nome-da-branch>
   ```

## 🧠 Exemplo prático

Para acessar a última aula (**com o código completo do perceptron clássico**):

```bash
git checkout implementando-um-perceptron-classico/exibindo-o-grafico-do-classificador-linear-binario
```

## 📌 Dica importante

Sempre que iniciar uma nova aula, faça o **checkout da branch correspondente**.  
Isso garante que você esteja trabalhando exatamente no mesmo ponto do conteúdo apresentado.
